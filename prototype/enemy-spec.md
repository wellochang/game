## 一、欄位說明

| 欄位      | 說明                                            |
|-----------|-------------------------------------------------|
| kind      | 使用的車輛 sprite（對應 CAR 物件）              |
| hp        | 最大 HP                                         |
| spd       | maxSpd（playerMaxSpd 同公式，但直接給定）       |
| arm       | 裝甲值，影響受到傷害                            |
| wRange    | 武器射程（gap-px），地圖情報顯示用              |
| wZoneRange| 武器 zone 射程，決定開火條件                    |
| wCd       | 武器冷卻（秒）                                  |
| wDmg      | 武器基礎傷害                                    |

## 五、選定邏輯

每個節點的敵人由種子 hash 決定（非每次隨機），同一張地圖中固定：

```js
const h = (map.seed + node.col × 13 + node.row × 7) % pool.length
enemy = pool[h]
```

## 七、敵方 AI

每回合（12 秒）結束時選擇新 zone（±1）：

```
diff = abs(playerZone - enemyZone)

if diff > wZoneRange:
    往玩家 zone 靠近 ±1   // 追入射程
else:
    維持當前 zone         // 在射程內，不動
```

開場 zone 由節點類型決定：

| 節點  | 玩家起始 | 敵方起始 |
|-------|----------|----------|
| fight | Z3       | Z2       |
| elite | Z3       | Z4       |
| boss  | Z3       | Z4       |
